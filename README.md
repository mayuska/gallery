# Gallery App

A simple Angular 21+ photo gallery app with infinite scroll and favorites.

## Features
- Infinite scroll: Loads more photos as you scroll down (custom implementation, no external libraries)
- Virtualized paging: Only the current and previous pages are rendered for performance
- Click a photo to add it to Favorites (saved in local storage)
- Material Design UI (Angular Material)

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```
3. Open [http://localhost:4200](http://localhost:4200) in your browser
